== Description ==

An ad management power tool plugin with many advanced advertising features to insert your ad by many positions on your website.

[Home](https://ads.azonow.com/) · [Documentation](https://ads.azonow.com/documentation/) · [Help & Support](https://my.azonow.com/support/) · [Pro Version](https://my.azonow.com/shop/azo-ads-pro/)

AZO Ads is the most comprehensive ad management tool available and is the most feature rich solution built for WordPress.

= Features =
* <strong>Cache Compatibility</strong>: AZO Ads works great with the most popular WordPress caching plugins: Autoptimize, Breeze Cache, Cache Enabler, GoDaddy Cache, Kinsta Cache, LiteSpeed Cache, Elementor (CSS cache), Premium Addons for Elementor, SiteGround SuperCacher, WP Fastest Cache, WP Optimize Cache, WP Rocket, WP Super Cache, W3 Total Cache, WPEngine Cache.
* <strong>Responsive Ad</strong> feature will make your ad looks good on all devices (PC, mobile).
* <strong>Beautiful UI/UX</strong> to help you manage the ad easily and quickly.
* <strong>Google Adsense</strong>: the plugin makes a deep integration with the ad types of Google Adsense such as Auto Ad, Display Ad, In-feed Ad & In-article Ad. You can adjust responsive or fixed sizes for your ad.
* Google Ad Manager (Double Click) support.
* <strong>15 Ad Positions</strong> will help you manage where your ad will show.
* <strong>10 Ad Network</strong> support.
* <strong>10 types condition of Visibility</strong>: 10 conditional ad visibility which helps you to control (include/exclude) on which posts/pages would you like the ads to appear.
* <strong>9 types condition of Targeting</strong> which will help you to control when your ad should appear.
* Banner Ads support.
* Background Ads support.
* Text/HTML/Javascript support.
* Video Ads support.
* Popup Ads support (Pro version only).
* Loop Ads support (Pro version only).
* Random Ads support (Pro version only).
* Floating Ads support (Pro version only).
* Group Insertion support (Pro version only).
* Carousel Ads support (Pro version only).
* Parallax Ads support (Pro version only).
* Rotator Ads support (Pro version only).
* Half Screen Ads support (Pro version only).
* Skip Ads support (Pro version only).
* Mobile Parallax Fullscreen Ads support (Pro version only).
* <strong>Appearance Layout Options</strong> where you can set up Alignment, Margin, Padding, Ad Label, Ad Label Position of the ads.
* <strong>Ad Label</strong>: You can easily add your custom label, you can choose the position to show label above or below of the ad.
* <strong>Expiration</strong>: Set up expiration time for the individual ad.
* <strong>Show Up by Days</strong>: Select which days to show up the individual ad (Pro version only).
* Duplicate/Clone your ad with the same configuration by just one click.
* Amazing speed with searching ad by using AJAX technology.
* <strong>Reports</strong> where you can get the chart about all statistics of your total ads or just specific ad for weekly, monthly or yearly. Very strong!
* <strong>Low Server Resource</strong> - AZO Ads plugin is coded well. It takes very low resource from the server such as: memory and CPU. This makes your website loading fast and getting SEO improvement.
* Multiple language support.

== Installation ==

How to install the plugin and get it working?

= Using The WordPress Dashboard (Recommended Installation) =

1. Navigate to the 'Add New' in the Plugins dashboard.
2. Search for 'AZO Ads'.
3. Click 'Install Now'.
4. Activate AZO Ads on the Plugin dashboard.

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the Plugin dashboard.
2. Navigate to the 'Upload' area.
3. Select `azo-ads.zip` from your computer.
4. Click 'Install Now'.
5. Activate AZO Ads in the Plugin dashboard.

= Using FTP =

1. Download `azo-ads.zip`
2. Extract the `azo-ads` directory to your computer
3. Upload the `azo-ads` directory to the `/wp-content/plugins/` directory
4. Activate AZO Ads in the Plugin dashboard
